# Presets

This directory holds the rule-pack YAML files. The canonical README for the
repo — including shipped presets, install instructions, and the roadmap — is
in [the parent directory](../README.md).

## Files

- [`clawie-default.yaml`](clawie-default.yaml) — Clawie `chat` intent
  (Anthropic Messages API + OpenAI Chat Completions API). Matches containers
  named `clawie-chat`.

Each preset is a versioned Outcall ruleset reviewable as a PR.
